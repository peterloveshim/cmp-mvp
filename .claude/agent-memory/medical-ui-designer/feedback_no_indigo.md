인디고 계열(#4F46E5)과 뻔한 메디컬 블루(#1E40AF 계열)를 primary로 제안하지 말 것.

**Why:** 사용자가 "인디고를 너무 자주 제안한다"고 피드백. 프로젝트마다 맥락 없이 안전한 기본값을 반복하는 것을 원하지 않음. 독창적이면서 도메인 맥락에 설득력 있는 색상을 원함.

**How to apply:** 색상 팔레트 제안 시 반드시 프로젝트 도메인(예: CMP = 환자 회송, 연결, 안전한 도착)의 의미와 색의 상징을 연결해서 논리적 근거를 제시할 것. 단순히 "신뢰감 = 블루" 공식 금지.
